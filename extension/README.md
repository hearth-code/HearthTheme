# HearthCode

Code by Hearthlight.

HearthCode is a warm, low-glare VS Code color theme set built for long coding sessions.
It keeps semantic token roles consistent across dark and light variants, so syntax cues stay stable when switching modes.

## Included Themes

- `Hearth Dark` (default)
- `Hearth Dark Soft`
- `Hearth Light`

## Why Teams Use HearthCode

- Long-session comfort: warm palette and controlled saturation
- Strong hierarchy: keywords, functions, strings, and types stay easy to scan
- Cross-mode consistency: Dark, Dark Soft, and Light share role semantics
- Production-ready maintenance: fixture-based screenshots and automated quality audits

## Install

- Marketplace: <https://marketplace.visualstudio.com/items?itemName=hearth-code.hearth-theme>
- VS Code Quick Open:

```text
ext install hearth-code.hearth-theme
```
